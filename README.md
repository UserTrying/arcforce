⚔️ ARCFORCE

A fantasy-military browser platformer shooter. Fight solo, team up, or go head-to-head — all in your browser, no downloads needed.

🎮 Play Now

👉 Play ARCFORCE

🕹️ Game Modes

Mode Description 🧍 Solo Survive endless waves of enemies. How long can you last? 🤝 Co-op Team up with a friend on the same keyboard and fight together ⚔️ PvP Go head-to-head against another player — last one standing wins

🎯 Controls

Player 1

Action Key Move Left / Right A / D Jump W Shoot Space

Player 2

Action Key Move Left / Right ← / → Jump ↑ Shoot Enter

🏆 Leaderboard

Scores are submitted automatically when a run ends. No account needed — just enter your name and your score is saved globally.

🌐 Built With





HTML5 Canvas



Vanilla JavaScript



CSS3

🛠️ Run Locally

git clone https://github.com/UserTrying/arcforce.git
cd arcforce
# Open index.html in your browser


No installs. No servers. Just open the file.
